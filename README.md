Estou atualmente estudando e aprendendo as mais recentes tecnologias front end, com foco em aprimorar minhas habilidades em React JS. Estou animado com o futuro como desenvolvedor web e ansioso para continuar aprendendo e crescendo nesta área emocionante e em constante evolução.
